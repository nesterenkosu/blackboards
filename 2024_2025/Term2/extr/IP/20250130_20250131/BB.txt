HTML
~~~~
HyperText Markup Language - язык разметки гипертекста - формирование структуры документа

CSS
~~~
Cascade Style Sheet - каскадная таблица стилей - для управления дизайном элементов документа

id - уникальный в рамках документа идентификатор некоторой одной HTML-сущности
class - уникальный идентификатор множества HTML-сущностей

# - селектор id
. - селектор class


Есть 2 специализированных тега, с которыми не ассоциировано никакое оформление и семантика (сами по себе эти теги не видоизменяют шрифт, и не несут специального значения для поисковых систем), они позволяют обозначить блок текста, над которым будут выполнены некоторые операции с помощью CSS или JavaScript.

<div>...</div> - блочный элемент (block). Занимает всю доступную ширину окна браузера.
<span>...</span> - строчный элемент (подстановочный, inline). Ширина равна строго ширине содержимого

Остальные теги HTML также могут себя вести либо как блочные, либо как строчные. Поведение может быть изменено с одного на другое с помощью CSS-свойства display.
display: block; блочный
display: inline; строчный
